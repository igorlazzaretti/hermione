Instalação Rápida

Escolha seu ambiente preferido e configure em segundos.

terminal

Instalação Global no Sistema

Após baixar, mova o arquivo hermione.sh

sudo mv hermione.sh /usr/local/bin/hermione

Dê permissão para o script

sudo chmod +x /usr/local/bin/hermione

Pronto! Basta digitar hermione

hermione

Editando o Script

Personalize a Hermione com seus próprios feitiços.

edit

Editando com Vim ou Sublime Text

Usando Vim

sudo vim /usr/local/bin/hermione

Para substituir o conteúdo:

  • Digite ggdG para apagar tudo.
  • Pressione i e cole o novo código.
  • Pressione Esc, digite :wq e Enter para salvar.

Usando Sublime Text

sudo subl /usr/local/bin/hermione
