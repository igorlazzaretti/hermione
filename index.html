Instalação Rápida

Escolha seu ambiente preferido e configure em segundos.

setup-vim.sh
# Instalar Hermione para Vim
$ curl -sSL https://hermione.sh/vim/install | bash
# Configurar atalhos no .vimrc
$ vim-hermione --setup-aliases
sublime-config.sh
# Baixar pacote Sublime
$ wget https://hermione.sh/sublime/plugin.zip
# Mover para diretório de Packages
$ unzip plugin.zip -d ~/.config/sublime-text/Packages/
terminal

Instalação Global no Sistema

Mova o arquivo hermione.sh

sudo mv hermione.sh /usr/local/bin/hermione

Dê permissão para o script

sudo chmod +x /usr/local/bin/hermione

Pronto! Basta digitar hermione
